# Tanalog(タナログ)
飲食店向けの在庫情報を一元管理するためのiOSアプリです。(現在はiPadのみ対応)

## 主な機能
### 1. 在庫情報
- 在庫名
- 個数（個数に応じて3色で色分け表示）
- タグ
### 2. 検索機能
- ひらがな、カタカナ、漢字に対応
### 3. 絞り込み機能
- タグで絞り込み
### 3. 並び替え機能
- 名前順
- 個数順
- 追加日

## 実装予定
- メニュー情報
- 発注情報
- 仕入れ先情報
